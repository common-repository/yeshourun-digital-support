Yeshourun Digital Support provides access to support engineers, allowing them to perform website maintenance & design.

== Description ==

Yeshourun Digital Support is an easy-to-use solution for free website support by providing access to support engineers who will perform regular website maintenance by updating plugins, themes, core, and prevent hacking by making sure all scripts on the website are up-to-date.

= Docs and support =

You can find [help](https://help.yeshourun.com), [FAQ](https://yeshourun.com/plugin/#faq) and more detailed information about Yeshourun Digital on [yeshourun.com](https://yeshourun.com/). When you can't find the answer to your question on the FAQ or in any of the documentation, check the [support forum](https://wordpress.org/support/plugin/yeshourun-digital-support/) on WordPress.org. If you can't locate any topics that pertain to your particular issue, post a new topic for it. Please keep in mind, the Wordpress.org support forum is ONLY for issues pertaining to the functioning of the plugin. All service requests should be submitted in the [app](https://app.yeshourun.com/service-requests/submit).

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* write any user personal data to the database;
* send any data to external servers;
* use cookies.

If you activate certain features in this plugin, access to your website may be granted to external parties. "Parties" meaning Yeshourun Digital and its support engineers. Thus, confirming our privacy policy is recommended.

* Yeshourun Digital ([Yeshourun Digital Wordpress Support](https://yeshourun.com/privacy-policy/))

== Installation ==

= Access to backend (wp-admin) required, intermediate difficulty =

1. Head to the **Add New Plugins** screen (**Plugins > Add New**).
2. Upload the entire archived `yeshourun-digital-support.zip` by clicking the **Upload Plugin** button on the right of the heading and choosing the file > Install Now.
3. Activate Plugin
4. You will find the **YD Support** menu in your WordPress admin screen.
5. Grant support access by clicking **Grant Access** in the plugin settings.

= Access to file manager required, mediocre difficulty =

1. Upload the entire unarchived `Yeshourun-Digital-Support` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
3. You will find the **YD Support** menu in your WordPress admin screen.
4. Grant support access by clicking **Grant Access** in the plugin settings.

For basic usage, have a look at the [plugin's docs](https://help.yeshourun.com/).

If you're still having difficulty activating the plugin, get support by [live chatting](https://yeshourun.com/). (Blue chat button on the bottom right-hand corner.)
